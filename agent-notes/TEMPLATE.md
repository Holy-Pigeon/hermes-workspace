## 经验主题：<一句话标题>

### 一句话结论
这条经验最核心、可立刻执行的结论。先给答案，不绕弯。

### 场景 / 触发
什么情况下会撞上这个问题？我当时在做什么、用的什么 agent / 模型 / 工具？

### 机制原理
从 agent 底层机制把"为什么会这样"讲清楚——上下文窗口与截断、agent loop 的调度与终止、工具调用协议、记忆的读写时机、多智能体的上下文隔离等。这是本栏目的灵魂：不止记"怎么做"，更要记"为什么"。

### 我的做法
具体怎么处理的。带可直接复制的配置 / prompt 片段 / 代码 / 命令。

### 反直觉点 / 踩过的坑
直觉上以为 X，实际是 Y。哪里翻过车、什么假设是错的。

### 适用边界
这招什么时候有效、什么时候失效。模型/框架/版本相关性。提醒未来的自己别过度外推。

### 关联
和哪些其他经验、机制、栏目条目相关。
