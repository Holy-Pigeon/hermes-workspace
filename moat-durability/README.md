# 护城河耐久度计分卡 (moat-durability)

把每篇研究 note 都无限延期的「护城河尽调」从定性故事翻译成**可量化、可证伪的耐久度数字**。

## 为什么存在
系统已有 8 个维度(价格/财报/估值分位/相关性/筹码/南向资金/反向DCF/发现管线), 但「护城河有多强、多耐久」从无任何工具量化。护城河 ≠ 故事——Buffett 的硬标准是**长期可持续的高 ROE + 稳定的 margin(定价权)**。

## 四支柱 (8年年报序列, 一手 akshare)
1. **ROE 持久性**: 过去 N 年 ROE≥15% 的年数占比 (Buffett 门槛)
2. **ROE 水平**: 近 N 年 ROE 中位数
3. **利润率稳定**: 销售净利率变异系数 CV=std/mean (越低=定价权越稳)
4. **利润率趋势**: 近半段 vs 远半段净利率均值差 (扩张=护城河加宽 / 收缩=被侵蚀)
辅助: 毛利率水平(financial_abstract兜底), 存货周转趋势。

## 判定 (纯启发式, 非买卖指令)
- 🏰 宽护城河: ROE≥15%年占比≥70% 且 净利率CV≤0.20
- 🧱 窄护城河: ROE≥15%年占比≥40% 或 净利率CV≤0.30
- ⚠️ 护城河存疑: 利润率收缩>3pp 或 ROE持久性<40%
- **定价权护栏**: 净利率<8% 的高ROE = 杠杆/周转驱动(薄利代工), 强制把 🏰 降级 🧱 并标注成因
- **低谷穿越升级**: 全样本判定保守(🧱/⚠️)、但近5年窗口同时满足 [ROE≥15%持久≥80% + 净利率斜率>0.3pp/年 + CV≤0.30 + 净利率均值≥8% + 非真侵蚀] 时升一级。修复全样本 CV/趋势对「上市初margin虚高→全行业结构性低谷→再上台阶」型公司(如宁德)的系统性假阴性: 远半含上市初虚高margin当基准会把真实margin扩张误读成趋势平/被侵蚀, 全行业低谷年会被算成个股单独失去定价权。CV≤0.30 护栏排除周期股(紫金CV0.56=金价红利非定价权, 不误升)。升级标注须核margin修复是否含成本端顺风(详见 research/note_2026-06-13_catl_moat_understated.md)。

## 已知口径警示
- **小商品城(600415)** 合并净利率受零毛利贸易流水(商品销售50%营收/1%毛利)稀释, 工具读到的「利润率收缩-4.9pp」可能是贸易流水占比上升的统计假象, 非真实护城河侵蚀。该标的须用 **市场经营分部毛利率(82%)** 而非合并口径判护城河(详见 research/note_2026-06-12_yiwu_lowpe_quality.md)。
- 港股(康方)无此端口, 不覆盖。
- 紫金等周期股净利率CV天然高(商品价驱动), 高CV≠无护城河, 是商品业务的本质。

## 数据诚实
ROE/净利率/周转全取 `stock_financial_analysis_indicator` 年报(12-31)行一手; 毛利率回退 `financial_abstract`; 取不到标 n/a 绝不编造。CV/趋势是历史会计读数不预测未来; 护城河会被技术/竞争颠覆, 高分≠该买。
