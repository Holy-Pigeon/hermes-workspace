## 今日知识点：向量投影（Vector Projection）——把一个向量在另一个方向上的"影子"量出来

### 所属类目
类目 1：线性代数基础

### 问题入口
你已经学过点积：$\mathbf{a} \cdot \mathbf{b}$ 能告诉你两个向量"有多像"。但有一个更具体的问题它没回答：

> 给定一个向量 $\mathbf{a}$，它在另一个向量 $\mathbf{b}$ 的方向上"占了多少"？

这不是抽象的好奇。注意力机制做的事，本质就是"把一个 token 的信息，按它在每个方向上的分量重新组合"。词向量去掉某个语义方向（比如"性别"方向）做去偏，也要先量出"在这个方向上有多少"。物理里把力分解到斜面方向、机器学习里把数据投到主成分方向（PCA）——全是同一个动作：**投影**。

所以问题很朴素：怎么把"一个向量在某个方向上有多少"算成一个确定的数，再算成一个确定的向量？

### 朴素想法
最自然的想法是用画图加直角三角形。把 $\mathbf{a}$ 和 $\mathbf{b}$ 画在同一起点，从 $\mathbf{a}$ 的箭头尖端向 $\mathbf{b}$ 所在的直线作一条垂线，垂足到原点的那段，就是"$\mathbf{a}$ 在 $\mathbf{b}$ 方向上的影子"。

如果你还记得三角函数，会写出：影子长度 $= |\mathbf{a}| \cos\theta$，其中 $\theta$ 是两个向量的夹角。这在二维、用尺子量角度时完全够用。

### 卡住的地方
朴素办法有两个致命问题：

1. **维度一高就画不出来。** 词向量是 768 维、4096 维，你没法"作垂线"，也没有现成的角度 $\theta$ 可量。$\cos\theta$ 本身还得先算出来——可你不知道怎么算。
2. **它只给了长度，没给方向。** "影子长 3"还不够，下游往往需要那个影子向量本身（带方向），比如"把 $\mathbf{a}$ 里属于 $\mathbf{b}$ 方向的部分减掉"。

关键的跃迁是：**点积恰好等于 $|\mathbf{a}||\mathbf{b}|\cos\theta$**，于是 $\cos\theta$ 不用量角器，用坐标算的点积就能反推出来。投影从"几何作图"变成了"纯代数公式"——这一步让它在任意高维都能算。

### 1. 概念

**标量投影（scalar projection）**：$\mathbf{a}$ 在 $\mathbf{b}$ 方向上的影子长度（带正负号）。记作 $\text{comp}_{\mathbf{b}}\mathbf{a}$。

**向量投影（vector projection）**：那个影子本身——一个沿着 $\mathbf{b}$ 方向的向量。记作 $\text{proj}_{\mathbf{b}}\mathbf{a}$。

正式表达。先回忆点积的几何定义：

$$
\mathbf{a} \cdot \mathbf{b} = |\mathbf{a}|\,|\mathbf{b}|\cos\theta
$$

其中 $|\mathbf{a}|$ 是 $\mathbf{a}$ 的长度（范数），$\theta$ 是两向量夹角。两边除以 $|\mathbf{b}|$：

$$
\text{comp}_{\mathbf{b}}\mathbf{a} = |\mathbf{a}|\cos\theta = \frac{\mathbf{a} \cdot \mathbf{b}}{|\mathbf{b}|}
$$

这就是标量投影：不用角度，只用点积和长度。再把它乘上 $\mathbf{b}$ 的**单位方向向量** $\frac{\mathbf{b}}{|\mathbf{b}|}$，就得到向量投影：

$$
\text{proj}_{\mathbf{b}}\mathbf{a} = \left(\frac{\mathbf{a} \cdot \mathbf{b}}{|\mathbf{b}|}\right)\frac{\mathbf{b}}{|\mathbf{b}|} = \frac{\mathbf{a} \cdot \mathbf{b}}{\mathbf{b} \cdot \mathbf{b}}\,\mathbf{b}
$$

这里 $\mathbf{b} \cdot \mathbf{b} = |\mathbf{b}|^2$，所以分母写成点积形式更对称、也更好编程。

通俗理解：太阳正对着 $\mathbf{b}$ 这条路垂直照下来，$\mathbf{a}$ 这根棍子斜插在地上，它在路面上投下的影子就是 $\text{proj}_{\mathbf{b}}\mathbf{a}$。影子的长度由"棍子多长"和"棍子和路的夹角"共同决定——但类比到此为止，真正算的时候我们不量角度，而是用上面那个点积公式。

### 2. 推导与证明

我们来证明：向量投影 $\text{proj}_{\mathbf{b}}\mathbf{a}$ 确实是"$\mathbf{a}$ 在 $\mathbf{b}$ 方向上、且使残差垂直于 $\mathbf{b}$"的那个向量。这才是投影的本质定义（最近点/正交分解）。

关键步骤：

1. **设投影是 $\mathbf{b}$ 的某个倍数。** 因为影子必须躺在 $\mathbf{b}$ 的方向上，所以它一定能写成 $\mathbf{p} = k\mathbf{b}$，其中 $k$ 是待定的标量。为什么这么设？因为"在 $\mathbf{b}$ 方向上"用代数语言说就是"和 $\mathbf{b}$ 平行"，平行就是标量倍数。

2. **要求残差垂直。** 把 $\mathbf{a}$ 拆成"影子 + 剩下的"：$\mathbf{a} = \mathbf{p} + \mathbf{r}$，残差 $\mathbf{r} = \mathbf{a} - k\mathbf{b}$。投影的几何含义是垂足，所以残差必须和 $\mathbf{b}$ 垂直，即 $\mathbf{r} \cdot \mathbf{b} = 0$。垂直为什么用点积等于 0？因为 $\cos 90^\circ = 0$，由点积几何定义立即得到。

3. **解出 $k$。** 把 $\mathbf{r} = \mathbf{a} - k\mathbf{b}$ 代入垂直条件：

$$
(\mathbf{a} - k\mathbf{b}) \cdot \mathbf{b} = 0 \quad\Rightarrow\quad \mathbf{a}\cdot\mathbf{b} - k\,(\mathbf{b}\cdot\mathbf{b}) = 0
$$

$$
k = \frac{\mathbf{a}\cdot\mathbf{b}}{\mathbf{b}\cdot\mathbf{b}}
$$

代回 $\mathbf{p} = k\mathbf{b}$ 就得到前面的公式。证毕。这个推导比"作垂线"强的地方在于：它只用了点积，从没用过角度，所以**在任意维度都成立**。

补充：如果把 $\mathbf{b}$ 先归一化成单位向量 $\hat{\mathbf{u}} = \mathbf{b}/|\mathbf{b}|$，公式简化为 $\text{proj} = (\mathbf{a}\cdot\hat{\mathbf{u}})\,\hat{\mathbf{u}}$。再写成矩阵形式，$\text{proj} = \hat{\mathbf{u}}\hat{\mathbf{u}}^{T}\mathbf{a}$，那个 $\hat{\mathbf{u}}\hat{\mathbf{u}}^{T}$ 就是**投影矩阵** $P$，它满足一个漂亮性质 $P^2 = P$（投影两次等于投影一次——影子再照一遍还是它自己）。

### 3. 来源与历史动机
- 概念：向量投影是欧几里得几何与线性代数的基础内容，没有单篇"发明论文"。
- 关键人物：约西亚·吉布斯（Josiah Willard Gibbs）与奥利弗·亥维赛（Oliver Heaviside）在 1880 年代把"向量代数 + 点积"系统化，投影随之成为标准工具。
- 时间：现代向量记号约定型于 19 世纪末。
- 链接：可参考 MIT 18.06 Gilbert Strang《Linear Algebra》关于 Projections 的章节（ocw.mit.edu）。

历史背景：投影最早服务于物理（把力、速度分解到坐标轴）和测量。进入统计学后，**最小二乘法**被重新理解为"把观测向量投影到模型张成的子空间上"——高斯和勒让德 19 世纪初的最小二乘，本质就是一次正交投影。这个"投影 = 最优近似"的视角，后来贯穿了 PCA、回归、乃至神经网络里的线性层。

### 4. 核心用途

用途一：**注意力机制里的信息重组。** Self-Attention 算完权重后，输出是各个 Value 向量的加权和——可以理解为把 Query 携带的信息，按它与各 Key 的对齐程度（点积），重新分配到 Value 张成的方向上。点积打分这一步，就是标量投影思想的直接应用：衡量 Query 在每个 Key 方向上有多少分量。

用途二：**词向量去偏与语义编辑。** 想去掉词向量里的"性别"成分，就先确定一个性别方向 $\mathbf{g}$，再对每个词向量 $\mathbf{w}$ 减掉它在 $\mathbf{g}$ 上的投影 $\mathbf{w} - \text{proj}_{\mathbf{g}}\mathbf{w}$，剩下的就是与性别正交的部分。激活引导（activation steering）、概念擦除同理。

用途三：**降维与最小二乘（PCA / 线性回归）。** PCA 是把数据投影到方差最大的几个方向上；线性回归的拟合值是把目标向量投影到特征张成的子空间上。投影矩阵 $P = X(X^TX)^{-1}X^T$ 就是这个动作的封装——你已经在"逆矩阵"那节见过它的零件。

### 5. 反直觉点与常见误区
反直觉点：**投影通常会让向量变短，而且这种"压缩"不可逆。** 一旦把 $\mathbf{a}$ 投到 $\mathbf{b}$ 方向上，垂直于 $\mathbf{b}$ 的信息就彻底丢了——你没法从影子还原出原来的棍子。这正是投影矩阵 $P$ 不可逆、行列式为 0 的原因（它把整个空间压扁到一条线/一个子空间上），也呼应了你学过的"秩"：投影矩阵的秩等于被投到的子空间维数。

常见误区：
- **误区一：以为 $\text{proj}_{\mathbf{b}}\mathbf{a} = \text{proj}_{\mathbf{a}}\mathbf{b}$。** 投影不对称！"$\mathbf{a}$ 在 $\mathbf{b}$ 上的影子"和"$\mathbf{b}$ 在 $\mathbf{a}$ 上的影子"是两个不同向量，方向都不一样。对称的只有点积这个数。
- **误区二：以为投影到 $\mathbf{b}$ 需要 $\mathbf{b}$ 是单位向量。** 不需要。公式里的分母 $\mathbf{b}\cdot\mathbf{b}$ 已经自动把 $\mathbf{b}$ 的长度归一化掉了，投影结果只取决于 $\mathbf{b}$ 的**方向**，与 $\mathbf{b}$ 的长度无关。把 $\mathbf{b}$ 换成 $2\mathbf{b}$，投影结果一模一样。

### 6. 小实验
拿纸笔手算，二维即可：

设 $\mathbf{a} = [3, 4]$，$\mathbf{b} = [1, 0]$（就是 x 轴方向）。

- 点积：$\mathbf{a}\cdot\mathbf{b} = 3\times1 + 4\times0 = 3$。
- $\mathbf{b}\cdot\mathbf{b} = 1$。
- 向量投影：$\text{proj}_{\mathbf{b}}\mathbf{a} = \frac{3}{1}[1,0] = [3, 0]$。

符合直觉：$[3,4]$ 在 x 轴上的影子正是 $[3,0]$。再验证残差垂直：$\mathbf{r} = [3,4]-[3,0] = [0,4]$，$\mathbf{r}\cdot\mathbf{b} = 0$，垂直成立。

进阶一步：把 $\mathbf{b}$ 换成 $[2,0]$（同方向但长度翻倍），重算一遍——你会发现投影结果还是 $[3,0]$，亲手验证"投影只看方向不看长度"。

想用代码核对，三行就够：

```python
import numpy as np
a, b = np.array([3., 4.]), np.array([1., 0.])
print((a @ b) / (b @ b) * b)   # [3. 0.]
```

### 7. 批判性思考
局限性：投影到**单个**向量方向，丢掉的信息太多。真实任务里我们几乎总是投影到一个**子空间**（多个基向量张成），这时若基向量彼此不正交，简单逐个投影再相加是错的，必须用投影矩阵 $X(X^TX)^{-1}X^T$ 或先做正交化（Gram-Schmidt / QR）。"逐方向投影可加"只在正交基下成立——这也是为什么正交性（你上一阶段学过）这么受偏爱。

深度问题：注意力的输出"Value 加权和"真的等价于一次投影吗？严格说不完全是——投影要求权重来自正交分解且和为可正可负，而注意力权重经过 Softmax 后非负且和为 1（是一个凸组合，落在 Value 们围成的凸包内）。想一想：这个"非负、和为 1"的约束，让注意力比纯线性投影多了什么、又少了什么表达能力？

### 8. 和后续知识的连接
- **缩放点积注意力**：QK 点积打分本质是在量"Query 在各 Key 方向上的分量"，投影视角能帮你理解为什么点积可以当对齐分数。
- **多头注意力 / 线性层**：每个 $W_Q, W_K, W_V$ 都是把输入投影到一个子空间，多头就是投到多个不同子空间各看一眼——你已学的"多子空间表示"正是投影的批量版。
- **RoPE 与正交矩阵**：旋转是保长度的正交变换，与"投影会丢长度"形成鲜明对比，两者一起构成理解注意力几何的左右手。
- **PCA / 表示分析 / 概念擦除**：日后做模型可解释性、激活引导时，"沿某方向投影、再减掉"会是你最常用的手术刀。
