## 今日知识点：范数（Norm）——给向量一把统一的"长度尺"

### 所属类目
类目 1：线性代数基础

### 问题入口
深度学习里到处都在说"这个向量大不大""两个向量差多远""梯度爆炸了"。可"大小"到底怎么量？一个词向量有 768 个分量，一层的权重矩阵有上百万个数——我们凭什么用一个数字说"它太大了，得压一压"？

更具体的问题：训练时我们想给模型加约束（别让权重无限膨胀）、想判断梯度有没有炸、想衡量两个嵌入向量像不像。这些操作的前提，是先把"一组数"塌缩成"一个可比较的标量"。这个把向量映射成一个非负实数的工具，就是**范数**。

### 朴素想法
最自然的想法：把向量所有分量加起来当"大小"。比如 $\mathbf{x} = [3, -4]$，加起来 $3 + (-4) = -1$。

或者退一步：取最大的那个分量。$[3, -4]$ 里最大是 $3$，就说它"有 3 那么大"。

### 卡住的地方
直接求和会出大问题：$[3, -4]$ 的和是 $-1$，但 $[-3, 4]$ 的和也是 $1$，而 $[100, -100]$ 的和是 $0$。一个明明分量很大的向量，"大小"居然是 $0$？正负相互抵消，让求和完全无法反映"这个向量离原点有多远"。

取最大分量也不够：$[3, -4]$ 和 $[3, 0]$ 的最大分量都是 $3$，但前者明显"更长"。只看一个分量，丢掉了其它维度的信息。

我们真正想要的"大小"，必须满足几条朴素直觉：① 永远非负，只有零向量大小为 $0$；② 把向量拉长 2 倍，大小也变 2 倍；③ 两个向量相加，长度不会超过各自长度之和（走斜边不会比绕两条直角边更远）。范数就是把这几条直觉公理化后得到的工具。

### 1. 概念
**范数**是一个函数，把向量 $\mathbf{x}$ 映射成一个非负实数 $\|\mathbf{x}\|$，并满足三条公理：

1. **非负性**：$\|\mathbf{x}\| \geq 0$，且 $\|\mathbf{x}\| = 0$ 当且仅当 $\mathbf{x} = \mathbf{0}$。
2. **齐次性**：$\|\alpha \mathbf{x}\| = |\alpha| \cdot \|\mathbf{x}\|$，其中 $\alpha$ 是任意标量。
3. **三角不等式**：$\|\mathbf{x} + \mathbf{y}\| \leq \|\mathbf{x}\| + \|\mathbf{y}\|$。

最常用的是 **$L_p$ 范数**家族：

$$
\|\mathbf{x}\|_p = \left( \sum_{i=1}^{n} |x_i|^p \right)^{1/p}
$$

这里 $x_i$ 是向量的第 $i$ 个分量，$n$ 是维数，$p \geq 1$ 是一个参数。换不同的 $p$，就得到不同的"量长度的方式"：

- $p = 2$：**$L_2$ 范数**（欧几里得范数），$\|\mathbf{x}\|_2 = \sqrt{\sum_i x_i^2}$，就是中学学的"勾股定理距离"。
- $p = 1$：**$L_1$ 范数**（曼哈顿范数），$\|\mathbf{x}\|_1 = \sum_i |x_i|$，像出租车只能沿街区横平竖直地走。
- $p \rightarrow \infty$：**$L_\infty$ 范数**（最大范数），$\|\mathbf{x}\|_\infty = \max_i |x_i|$，只看绝对值最大的那个分量。

通俗理解：范数就像不同城市量"两点距离"的规矩。欧几里得范数是"直升机能直线飞"的距离；$L_1$ 是"出租车必须沿马路走"的距离；$L_\infty$ 是"只关心哪条街最长"。它们都叫"距离"，但量出来的数不一样——而且都比"分量直接求和"靠谱，因为先取了绝对值（或平方），正负不再抵消。

### 2. 推导与证明
我们来验证 $L_2$ 范数确实满足三条公理，重点证最不显然的**三角不等式**，并解释为什么求和法不是范数。

关键步骤：

1. **为什么求和不是范数**：定义 $f(\mathbf{x}) = \sum_i x_i$。取 $\mathbf{x} = [1, -1]$，则 $f(\mathbf{x}) = 0$，但 $\mathbf{x} \neq \mathbf{0}$，违反非负性公理①（只有零向量才能为 $0$）。所以"直接求和"根本不配叫范数。修复方法就是先让每个分量变非负——取绝对值得到 $L_1$，取平方再开根得到 $L_2$。

2. **$L_2$ 的非负性与齐次性**：$\|\mathbf{x}\|_2 = \sqrt{\sum_i x_i^2}$，平方和恒非负，开根也非负；只有所有 $x_i = 0$ 时才为 $0$，满足公理①。齐次性：$\|\alpha\mathbf{x}\|_2 = \sqrt{\sum_i (\alpha x_i)^2} = \sqrt{\alpha^2 \sum_i x_i^2} = |\alpha|\sqrt{\sum_i x_i^2} = |\alpha|\|\mathbf{x}\|_2$，满足公理②。

3. **三角不等式的核心——柯西-施瓦茨不等式**：要证 $\|\mathbf{x}+\mathbf{y}\|_2 \leq \|\mathbf{x}\|_2 + \|\mathbf{y}\|_2$，先用一个关键引理。柯西-施瓦茨不等式说：两个向量的点积绝对值不超过各自长度之积，

$$
|\mathbf{x} \cdot \mathbf{y}| = \left| \sum_i x_i y_i \right| \leq \|\mathbf{x}\|_2 \, \|\mathbf{y}\|_2
$$

直觉上它对应 $\mathbf{x}\cdot\mathbf{y} = \|\mathbf{x}\|\|\mathbf{y}\|\cos\theta$，而 $|\cos\theta| \leq 1$。

4. **拼出三角不等式**：把 $\|\mathbf{x}+\mathbf{y}\|_2^2$ 展开，
$$
\|\mathbf{x}+\mathbf{y}\|_2^2 = \sum_i (x_i+y_i)^2 = \|\mathbf{x}\|_2^2 + 2\,(\mathbf{x}\cdot\mathbf{y}) + \|\mathbf{y}\|_2^2
$$
用第 3 步把中间项放大：$\mathbf{x}\cdot\mathbf{y} \leq \|\mathbf{x}\|_2\|\mathbf{y}\|_2$，于是
$$
\|\mathbf{x}+\mathbf{y}\|_2^2 \leq \|\mathbf{x}\|_2^2 + 2\|\mathbf{x}\|_2\|\mathbf{y}\|_2 + \|\mathbf{y}\|_2^2 = \big(\|\mathbf{x}\|_2 + \|\mathbf{y}\|_2\big)^2
$$
两边开根（都非负）即得 $\|\mathbf{x}+\mathbf{y}\|_2 \leq \|\mathbf{x}\|_2 + \|\mathbf{y}\|_2$。三条公理全部满足，$L_2$ 是合法范数。证毕。

每一步的"为什么"：先取平方是为了消掉正负号、让齐次性以 $|\alpha|$ 形式出现；引入柯西-施瓦茨是因为三角不等式的难点恰好卡在"两个向量的交叉项"上，而这个引理正好给交叉项一个上界。

### 3. 来源与历史动机
- 论文/著作：范数的公理化源于 20 世纪初泛函分析的兴起，由 Stefan Banach 在其 1922 年博士论文及后续工作中系统化（"Banach 空间"即赋范完备空间）。
- 关键人物：Stefan Banach（巴拿赫）、David Hilbert（希尔伯特，$L_2$ 内积空间）、Hermann Minkowski（闵可夫斯基，三角不等式在 $L_p$ 下的推广即"闵可夫斯基不等式"）。
- 时间：$L_2$ 距离可追溯到古希腊勾股定理；$L_p$ 范数与闵可夫斯基不等式约 1900 年前后成形；公理化的赋范空间理论约 1920 年代确立。
- 链接：可参阅泛函分析标准教材（如 Rudin《Functional Analysis》）或 Wikipedia "Norm (mathematics)" 词条。

历史背景：数学家需要把"长度""距离""收敛"从二维三维几何推广到无限维函数空间（比如比较两个函数差多远）。直接搬勾股定理行不通，于是抽象出三条公理：凡满足这三条的函数都叫范数。这一抽象让微积分、概率、优化在统一的"赋范空间"框架下展开——深度学习的损失函数、正则项本质上都活在这个框架里。

### 4. 核心用途
用途一：**衡量误差与相似度**。回归任务的均方误差本质是预测向量与真值向量差的 $L_2$ 范数平方；推荐系统、检索里用 $L_2$ 距离或基于范数归一化的余弦相似度判断两个嵌入像不像。实际场景：句向量检索时先把每个向量除以它的 $L_2$ 范数做归一化，再算点积。

用途二：**正则化，约束模型复杂度**。$L_2$ 正则（权重衰减）在损失里加 $\lambda\|\mathbf{w}\|_2^2$，逼权重整体变小、更平滑；$L_1$ 正则加 $\lambda\|\mathbf{w}\|_1$，因为它的"棱角"几何特性会把许多权重精确压到 $0$，产生稀疏模型。实际场景：几乎所有 Transformer 训练都开权重衰减防止过拟合。

用途三：**监控与稳定训练**。梯度裁剪（gradient clipping）先算梯度向量的 $L_2$ 范数，一旦超过阈值就整体按比例缩小，防止梯度爆炸；LayerNorm/RMSNorm 用范数把激活向量缩放到稳定尺度。实际场景：训练大语言模型时几乎必开梯度裁剪，阈值常设在 $1.0$ 附近。

### 5. 反直觉点与常见误区
反直觉点：**同一个向量，换个范数"长度"可以差很多，而且"哪个更长"的排名都可能反转**。对 $\mathbf{x}=[1,1,1,1]$，$\|\mathbf{x}\|_1 = 4$、$\|\mathbf{x}\|_2 = 2$、$\|\mathbf{x}\|_\infty = 1$。维数越高，$L_1$ 和 $L_2$、$L_\infty$ 的差距越被放大——这正是"高维空间里距离行为很怪"的根源之一，也是为什么有人说高维下 $L_1$ 有时比 $L_2$ 更适合做距离度量。

常见误区：
- 误区一：把 $L_2$ 范数和它的平方混为一谈。机器学习里写损失常用 $\|\mathbf{w}\|_2^2$（不带根号），因为平方处处可导、求梯度干净；而 $\|\mathbf{w}\|_2$ 在原点不可导。两者最小化方向一致但不是同一个函数，写公式时别漏掉平方。
- 误区二：以为"$L_1$ 让权重变小、$L_2$ 也让权重变小，所以效果差不多"。实际上 $L_1$ 制造**稀疏**（很多权重精确为 $0$），$L_2$ 只是让权重**整体变小但很少恰好为 $0$**。差别来自它们等高线的几何形状（$L_1$ 是带尖角的菱形，尖角更容易和损失等高线相切在坐标轴上）。

### 6. 小实验
拿出纸笔，对 $\mathbf{x} = [3, 4]$ 手算三种范数：

- $L_2$：$\sqrt{3^2 + 4^2} = \sqrt{9+16} = \sqrt{25} = 5$。
- $L_1$：$|3| + |4| = 7$。
- $L_\infty$：$\max(3, 4) = 4$。

会发现恒有 $\|\mathbf{x}\|_\infty \leq \|\mathbf{x}\|_2 \leq \|\mathbf{x}\|_1$（这里 $4 \leq 5 \leq 7$）。再验证三角不等式：取 $\mathbf{y}=[1,0]$，$\mathbf{x}+\mathbf{y}=[4,4]$，$\|\mathbf{x}+\mathbf{y}\|_2 = \sqrt{32}\approx 5.66$，而 $\|\mathbf{x}\|_2 + \|\mathbf{y}\|_2 = 5 + 1 = 6$，确实 $5.66 \leq 6$，斜边没超过两边之和。

想用代码验证可跑：

```python
import numpy as np
x = np.array([3.0, 4.0])
print(np.linalg.norm(x, 2))    # 5.0
print(np.linalg.norm(x, 1))    # 7.0
print(np.linalg.norm(x, np.inf))  # 4.0
```

### 7. 批判性思考
局限性：范数把整个向量塌缩成一个标量，必然丢信息——两个范数相等的向量可以指向完全不同的方向。所以单看范数无法判断"方向像不像"，相似度任务还得配合点积或夹角。此外 $L_p$（$p<1$）不再满足三角不等式，严格说不是范数，用它做"距离"要格外小心。

深度问题：为什么深度学习里 $L_2$ 几乎是默认选择，而不是 $L_1$ 或 $L_\infty$？提示从三个角度想——可导性（$L_2$ 的平方处处光滑，利于梯度优化）、旋转不变性（$L_2$ 在坐标系旋转下不变，符合"空间各向同性"的物理直觉）、以及与高斯分布/最小二乘的天然联系（最大似然在高斯噪声假设下恰好导出 $L_2$ 损失）。那么在什么情况下你会主动放弃 $L_2$ 改用 $L_1$？（答案线索：当你需要稀疏、或数据有大量离群点时——$L_1$ 对离群点不像 $L_2$ 那样被平方放大。）

### 8. 和后续知识的连接
范数是后面一长串概念的地基：
- **LayerNorm / RMSNorm / BatchNorm**：都在用范数（或其变体）把激活向量重新缩放到稳定尺度，已学的归一化课程本质就是"除以某种范数再平移"。
- **L1/L2 正则化**：已学的正则化课程，其损失附加项就是权重向量的范数。
- **梯度裁剪**：训练 Transformer 防梯度爆炸，直接对梯度向量算 $L_2$ 范数。
- **注意力与嵌入相似度**：缩放点积注意力、余弦相似度都建立在向量长度（范数）之上；嵌入归一化就是把向量除以自身 $L_2$ 范数。
- **权重初始化与谱范数**：后续若涉及矩阵的"算子范数/谱范数"，正是把这里的向量范数推广到矩阵，用于分析网络的稳定性与 Lipschitz 性质。
