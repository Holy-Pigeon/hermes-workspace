## 今日知识点：协方差与相关系数（Covariance & Correlation）——衡量两个变量是否“一起动”的尺子

### 所属类目
类目 3：概率与统计

### 问题入口
我们已经会用**方差**衡量*一个*随机变量围着平均值晃动得有多厉害。但真实世界里，变量很少单独行动：气温升高时冰淇淋销量上涨；一个 token 的某个特征维度变大时，另一个维度往往也跟着变大。

一个非常具体的问题：**两个变量是不是“一起动”？是同涨同跌，还是一个涨另一个就跌？这种“联动”能不能用一个数字量出来？**

如果不能量化联动，我们就无法回答深度学习里一堆关键问题：词向量的不同维度是冗余的还是各管各的？做归一化时，为什么有时只减均值除标准差还不够，还要“白化”？PCA 凭什么能把高维数据压缩到几个方向？这些全都建立在“变量间如何联动”这一基石上。

### 朴素想法
最自然的想法：既然方差是 $\text{Var}(X) = E[(X - \mu_X)^2]$，也就是“$X$ 偏离自己均值的量，乘自己一遍，再取平均”，那衡量两个变量联动，我干脆**把两组数据画成散点图，用眼睛看像不像一条斜线**不就行了？

或者更省事：直接算两个变量的乘积平均 $E[XY]$，大就说明相关。

### 卡住的地方
- **用眼睛看**无法放进算法、无法比较强弱、更无法推广到 768 维词向量里的两两关系（那是上十万对关系，没法逐个看图）。
- **直接用 $E[XY]$** 会被“均值水平”污染。假设 $X, Y$ 都是很大的正数（比如都在 1000 附近），就算它们毫无联动，$E[XY]$ 也会是个大正数。你测到的是“它们都很大”，而不是“它们一起动”。
- 就算我们减掉均值，**$E[(X-\mu_X)(Y-\mu_Y)]$ 的数值还会受单位影响**：用“米”还是“厘米”量身高，算出来的数字差 100 倍，但联动的“真实强弱”根本没变。一个会随单位伸缩的数字，没法横向比较。

朴素想法卡在两点：**没去掉均值水平**、**没去掉量纲尺度**。协方差解决前者，相关系数解决后者。

### 1. 概念

**协方差（Covariance）** 衡量两个随机变量*共同偏离各自均值*的平均程度：

$$
\text{Cov}(X, Y) = E\big[(X - \mu_X)(Y - \mu_Y)\big]
$$

每个符号的含义：$X, Y$ 是两个随机变量；$\mu_X = E[X]$、$\mu_Y = E[Y]$ 是它们各自的均值（期望）；$E[\cdot]$ 表示对所有可能取值求平均。

直觉：把每个样本点相对各自均值的偏差 $(X-\mu_X)$ 和 $(Y-\mu_Y)$ 相乘——
- 如果两者**同时**大于均值（都正）或**同时**小于均值（都负），乘积为正 → 同涨同跌；
- 如果一个高于均值另一个低于均值，乘积为负 → 此消彼长；
- 把所有样本的这种“符号一致性”平均起来，正得越多说明越倾向同向，负得越多说明越倾向反向，接近 0 说明没有线性联动。

注意：当 $Y = X$ 时，$\text{Cov}(X, X) = E[(X-\mu_X)^2] = \text{Var}(X)$。**协方差是方差的推广**——方差是“变量和自己的协方差”。

**相关系数（Pearson Correlation）** 把协方差除以两个标准差，做成一个无量纲、范围被锁死在 $[-1, 1]$ 的标准化版本：

$$
\rho_{XY} = \frac{\text{Cov}(X, Y)}{\sigma_X \, \sigma_Y}
$$

其中 $\sigma_X = \sqrt{\text{Var}(X)}$ 是 $X$ 的标准差。$\rho = 1$ 表示完美正线性相关，$\rho = -1$ 完美负线性相关，$\rho = 0$ 无线性相关。

通俗理解：协方差像“两个人跳舞配合得好不好”的原始打分，但这个分受“舞台多大、用什么单位计分”影响；相关系数则是把分数归一化成“配合度百分比”，无论舞台多大都能横向比较。但要记住——它只测**线性**配合，舞步如果是非线性的复杂花样，它可能给出 0 分却完全误判（后面会展开）。

### 2. 推导与证明

我们来证明一个核心性质，也是它能成为“尺子”的根本原因：**相关系数一定落在 $[-1, 1]$ 之间**。这等价于柯西-施瓦茨不等式（Cauchy–Schwarz）。

关键步骤：

1. **构造一个一定非负的东西。** 令去均值后的变量 $\tilde{X} = X - \mu_X$、$\tilde{Y} = Y - \mu_Y$。考虑对任意实数 $t$ 的表达式：
$$
g(t) = E\big[(\tilde{X} - t\,\tilde{Y})^2\big]
$$
因为括号里是平方，恒 $\geq 0$，所以对任意 $t$ 都有 $g(t) \geq 0$。*为什么这么做*：平方的期望非负，这是个免费的、永远成立的约束，我们要把它“榨”出信息。

2. **展开成关于 $t$ 的二次函数。**
$$
g(t) = E[\tilde{X}^2] - 2t\,E[\tilde{X}\tilde{Y}] + t^2 E[\tilde{Y}^2] = \text{Var}(X) - 2t\,\text{Cov}(X,Y) + t^2 \text{Var}(Y)
$$
*为什么*：利用期望的线性性把平方拆开，每一项正好对应方差和协方差的定义。

3. **二次函数恒非负 ⟹ 判别式非正。** 一个开口向上（$\text{Var}(Y) \geq 0$）的二次函数 $a t^2 + b t + c \geq 0$ 对所有 $t$ 成立，要求判别式 $b^2 - 4ac \leq 0$。代入 $a=\text{Var}(Y)$、$b=-2\text{Cov}(X,Y)$、$c=\text{Var}(X)$：
$$
4\,\text{Cov}(X,Y)^2 - 4\,\text{Var}(X)\text{Var}(Y) \leq 0
$$
*为什么*：如果判别式为正，二次函数会有两个根、中间一段为负，与第 1 步“恒非负”矛盾。

4. **整理得到结论。**
$$
\text{Cov}(X,Y)^2 \leq \text{Var}(X)\,\text{Var}(Y) \quad\Longrightarrow\quad \frac{\text{Cov}(X,Y)^2}{\sigma_X^2 \sigma_Y^2} \leq 1 \quad\Longrightarrow\quad -1 \leq \rho_{XY} \leq 1
$$

证毕。而且取等号（$\rho = \pm 1$）当且仅当 $\tilde{X} - t\tilde{Y}$ 的平方期望为 0，即 $X$ 和 $Y$ 严格成线性关系 $X - \mu_X = t(Y - \mu_Y)$。**这就解释了为什么 $\rho = \pm 1$ 恰好对应“完美线性关系”。**

### 3. 来源与历史动机
- 概念：协方差与（积矩）相关系数
- 主要奠基者：Francis Galton（提出“相关/回归”的思想，1880s 研究遗传与身高），Karl Pearson（给出现代数学形式，即 Pearson product-moment correlation）
- 时间：约 1888–1896 年
- 链接：Pearson, K. (1896). "Mathematical Contributions to the Theory of Evolution. III. Regression, Heredity, and Panmixia." *Philosophical Transactions of the Royal Society*. 概念综述见 https://en.wikipedia.org/wiki/Covariance 与 https://en.wikipedia.org/wiki/Pearson_correlation_coefficient

历史背景：Galton 研究“父母身高与子女身高的关系”，发现高个父母的孩子平均偏高但会“向均值回归”，他需要一个数字刻画这种联动强度。Pearson 把它精确化成今天的公式。这套工具后来成了统计学、计量经济学、机器学习里描述变量关系的通用语言——没有它，就没有线性回归、PCA、乃至现代协方差矩阵的一整套机器学习理论。

### 4. 核心用途

用途一：**协方差矩阵与 PCA / 白化。** 把一个 $d$ 维随机向量 $\mathbf{x}$ 的所有维度两两协方差排成一张表，就是协方差矩阵 $\Sigma_{ij} = \text{Cov}(x_i, x_j)$（对角线是各维方差）。对它做特征值分解就是 PCA——找到数据“晃动”最猛的几个正交方向，从而降维。深度学习里对输入做“白化（whitening）”就是让协方差矩阵变成单位阵，去掉维度间冗余。这正好接上我们学过的**特征值分解**和**正定矩阵**（协方差矩阵一定是半正定的）。

用途二：**衡量特征冗余与表示质量。** 训练好的词向量或隐藏层，如果不同维度之间相关系数普遍很高，说明表示是冗余的、有效维度低；反之低相关说明每个维度都在“干自己的活”。这是分析表示坍缩（representation collapse）、对比学习里防止特征塌成一条线的关键诊断指标。

用途三：**理解归一化与初始化的统计基础。** 我们学过的 BatchNorm 减均值除标准差，本质是在“标准化每个维度的方差”；而 Xavier/He 初始化的推导，核心就是控制每层激活的方差不爆炸不消失。当涉及多维、维度间不独立时，方差的故事必须升级成协方差的故事。注意力里 $QK^\top$ 的缩放 $\frac{1}{\sqrt{d}}$ 之所以是那个值，也来自对点积方差的统计假设。

### 5. 反直觉点与常见误区

反直觉点：**相关系数为 0，绝不代表两个变量无关，只代表没有*线性*关系。** 经典例子：设 $X$ 在 $[-1,1]$ 上均匀分布，$Y = X^2$。$Y$ 完全由 $X$ 决定（最强的依赖关系！），但因为对称性，$\text{Cov}(X, X^2) = E[X^3] - E[X]E[X^2] = 0 - 0 = 0$，相关系数为 0。一个被它完全决定的变量，线性相关却测不出来。这是 Pearson 相关最大的陷阱。

常见误区：
- **误区一：把相关当因果。** “相关不蕴含因果”是统计学第一铁律。冰淇淋销量和溺水人数高度正相关，但凶手是“夏天高温”这个共同原因，不是冰淇淋。
- **误区二：以为协方差大就是关系强。** 协方差带量纲，数值大可能只是因为变量本身尺度大。比较强弱必须用归一化后的相关系数。
- **误区三：把“不相关”等同于“独立”。** 独立一定不相关（协方差为 0），但不相关推不出独立（上面 $Y=X^2$ 的例子就是反例）。只有在联合高斯分布这一特殊情形下，二者才等价。

### 6. 小实验
不用代码，手算 4 个样本即可看清符号机制。

设两组数据：身高偏差与体重偏差（已去均值）配对为：$(+2,+3),\ (+1,+1),\ (-1,-2),\ (-2,-2)$。

协方差（用样本平均，除以 4）：
$$
\text{Cov} = \frac{(2)(3) + (1)(1) + (-1)(-2) + (-2)(-2)}{4} = \frac{6 + 1 + 2 + 4}{4} = \frac{13}{4} = 3.25
$$

每一对乘积都是正的 → 高个偏重、矮个偏轻 → 同涨同跌 → 协方差为正，符合直觉。

现在把体重单位从“千克”换成“克”，所有体重偏差 $\times 1000$：协方差立刻变成 $3250$，暴涨 1000 倍。但你会发现**相关系数纹丝不动**（分子涨 1000 倍，分母里的 $\sigma_Y$ 也涨 1000 倍，约掉了）。亲手算一遍这两个数，你就彻底理解了“为什么需要相关系数来横向比较”。

进一步可心算验证反直觉点：取 $X = -1, 0, 1$（再加个 0 让均值为 0），$Y = X^2 = 1, 0, 1$。算 $\sum X_i (Y_i - \bar Y)$ 会发现正负相消为 0——强依赖、零相关，亲眼见证 Pearson 的盲区。

### 7. 批判性思考

局限性：Pearson 相关只能捕捉**线性**关系，对单调非线性（如指数、对数关系）和非单调关系都会低估甚至判零。它还对**异常值极其敏感**——一两个离群点能把相关系数从 0 拉到 0.9。实务中常用 Spearman 秩相关（看排序）或互信息（看任意依赖）来补足。

深度问题：既然相关系数有这么多盲区，为什么深度学习还能靠点积注意力（本质是一种内积/协方差式的相似度）work 得这么好？提示——Transformer 把原始 token 先经过可学习的线性变换 $W_Q, W_K$ 投影到新空间*再*做点积。换句话说，**模型自己学会了一个非线性特征空间，在那个空间里，线性相似度（点积）就足够表达原空间里的复杂关系**。线性工具之所以够用，是因为前面有非线性层（嵌入、FFN、激活）替它把空间“掰直”了。这正是“浅层用线性度量、深层堆非线性变换”这一深度学习核心范式的缩影。

### 8. 和后续知识的连接
- **特征值分解 / 正定矩阵**（已学）：协方差矩阵是半正定矩阵的最重要实例，PCA 就是对它做特征分解。
- **方差 / BatchNorm / Xavier·He 初始化**（已学）：方差的多维推广就是协方差矩阵，归一化与初始化的严格推导都要用到它。
- **缩放点积注意力**（已学）：$\frac{1}{\sqrt{d}}$ 缩放因子来自对点积方差的分析，点积本身就是一种未中心化的协方差式相似度。
- **词嵌入 / 表示坍缩**（已学/进阶）：用协方差矩阵诊断嵌入维度是否冗余、表示是否塌缩，是对比学习与表示学习的标准分析工具。
- 往后学到**协方差矩阵估计、卡尔曼滤波式的不确定性传播、扩散模型中的高斯噪声协方差**时，今天这把尺子还会反复出现。
