## 1. 硬数据（akshare 一手，财务摘要）

### 增长轨迹——清晰的逐季减速直到转负
| 报告期 | 营收YoY | 归母净利YoY |
|---|---|---|
| 2024H1 | +17.6% | +15.9% |
| 2024FY | +15.7% | +15.4% |
| 2025Q1 | +10.7% | +11.6% |
| 2025H1 | +9.2% | +8.9% |
| 2025Q3(累) | +6.3% | +6.2% |
| **2025FY** | **-1.2%** | **-4.5%** |
| 2026Q1 | +6.3% | +1.5% |

**两个关键事实**：
1. 减速是**连续单调**的（+15.7%→+10.7%→+9.2%→+6.3%→-1.2%），不是一次性脉冲 = 结构性需求降温，非偶发。
2. **2026Q1 营收已回正 +6.3%**（利润 +1.5% 滞后）—— 与迈瑞同型：营收先转正、利润滞后。但茅台减速的成因与迈瑞（反腐控费冻结）不同，须单独判。

### 盈利质量——margin 纹丝不动（关键判别点）
- 销售净利率：2023 52.5% / 2024 52.3% / **2025 50.5%** / 2026Q1 52.2% —— **CV 0.018（moat工具实测，全组合最稳）**
- 销售毛利率：~91%（基本不动）
- 摊薄ROE：2023 34.7% / 2024 37.0% / **2025 33.6%**

**机制判定**：-4.5% 的利润下滑 **几乎全部来自营收停滞（量/价），而非 margin 崩塌**。茅台没有打价格战、没有牺牲毛利去保量。这与：
- 紫金（margin 22.9%→36.3% 飙峰值=峰值幻觉陷阱）—— **机制相反**
- 金山（投资收益污染纸面利润）—— **机制无关**（茅台 TTM 现金含量正常，2026Q1 OCF/归母 98.8%）
- 迈瑞（反腐控费冻结去库存）—— **同为"营收先转正"型，但茅台需求降温是终端消费疲软+高端白酒去库存，非外部行政冻结**

### 反向验证（reverse_dcf, r=9% 因必需消费低β / N=5y）
现价 1291.91 / EPS-TTM 66.25 / PE 19.5x 隐含未来 5 年净利复合增速：
| 退出倍数 | 隐含 g |
|---|---|
| 18x | **10.8%** |
| 22x | 6.4% |
| 25x | 3.7% |

**关键反差**：现价（保守退出 18x）只 price-in **10.8% 复合增速**，而茅台过去 5 年实际 CAGR 远高于此（2020-2024 净利 CAGR ~17%）。即使按"新常态"假设茅台进入个位数增长时代，10.8% 的隐含门槛对一个 91% 毛利、50% 净利率、33% ROE、品牌不可复制的资产而言，**有显著安全垫**——前提是负增长是周期低谷而非永久趋势。这正是 reverse_dcf 在工业富联（隐含25-30%贴着实际无垫）对面的极端：茅台是"低预期+强资产"，工业富联是"高预期+脆弱"。

## 3. 可证伪论点

**主论点**：茅台 PE 史2%分位 + 首次负增长，是**高端消费周期下行+渠道去库存的低谷**，而非品牌护城河永久受损；史上最便宜估值 × 完好的 margin/ROE/毛利 = 错杀概率大于陷阱概率。

**证伪条件（绑 2026 中报 8/31 + 后续季度）**：
- **证伪为陷阱**：若 ① 净利率跌破 48%（茅台开始降价/促销保量=定价权受损=护城河实质裂痕）；或 ② 营收 YoY 再度转负（2026Q1 +6.3% 是反弹假象）；或 ③ 一批价（飞天茅台批价，需另查渠道数据）跌破出厂价持续倒挂（终端需求崩塌的硬信号）。
- **证实为低谷**：净利率守住 50%+ 且营收 YoY 维持正增长（即便个位数）+ 直营/i茅台占比提升对冲渠道压力。

## 4. 反方论据（抗确认偏误）

1. **"高端白酒黄金时代结束"的结构论**：年轻一代消费习惯转移、商务宴请缩减、健康意识、人口结构 → 若是这条，则茅台进入永久低增长，19.5x 对一个 0-5% 增长的公司可能就是该有的估值（非便宜）。这是最强的看空逻辑，无法被一两个季度证伪，须看 2-3 年量趋势。
2. **批价是真正的领先指标，本note缺失**：茅台的需求强弱最早反映在飞天茅台二级市场批价，而非财报。本note用财报滞后数据，**批价数据是最大盲区**，下一步必须补。
3. **直营渠道改革的双刃**：直营提毛利但也可能因控价/控量牺牲短期营收，2025 营收转负部分可能是主动调控而非纯需求 → 需拆分渠道结构。
4. **政策风险**：高端消费若遇新一轮"反浪费/反三公"政策收紧，茅台首当其冲（历史2012-2014塑化剂+八项规定时茅台股价腰斩，事后证明是黄金坑——但当时也真跌了2年）。

## 数据诚实声明

- 财务全 akshare 一手财务摘要（按报告期），TTM 为滚动加总。

- 候选 ≠ 买入，须批价坐实 + 渠道拆分 + 护城河永久性判断，非买卖指令。
