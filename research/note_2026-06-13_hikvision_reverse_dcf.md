## 0. 触发与本篇定位
上轮(08:30)修复发现管线现金含量护栏 TTM 口径后, 海康威视从"被单季 -80% 假红旗遮蔽"升为 ⭐价值成长候选, 并留下四条待验证。本篇推进其中三条最硬的:
- ① reverse_dcf 算 18.6x 隐含的真实增长预期是否合理(有无安全垫)
- ② 拆 26Q1 归母 +36% vs 营收 +12% 的剪刀差来源(是运营杠杆/margin 正常化, 还是一次性?)
- ③ 海外占比 + 制裁/关税敞口量化(地缘是压低倍数的主因, 必须量化而非定性)

## 1. 估值锚(一手核实)
- 现价 **30.24**(6/12 新浪收盘; 东财实时端口 RemoteDisconnected, 对 18.6x 量级无影响)
- EPS-TTM = 0.303(26Q1) + 1.546(25FY) − 0.221(25Q1) = **1.628**
- **PE-TTM = 30.24 / 1.628 = 18.6x**(与发现管线读数一致)
- 自身历史分位(上轮 valuation 通道): PE 史 6% / PB 3% / PS 8% 分位 = 史上最便宜区
- 年度 ROE 17.3%(25FY) / 毛利率 49.1% / PEG 0.9

## 2. 剪刀差归因(待验证②)——结论: 真实运营杠杆 + margin 正常化, 非一次性
| 指标 | 25Q1 | 26Q1 | YoY |
|---|---|---|---|
| 营业总收入(百万) | 18,531.9 | 20,715.4 | **+11.8%** |
| 归母净利(百万) | 2,038.7 | 2,781.2 | **+36.4%** |
| 扣非净利(百万) | 1,926.4 | 2,647.6 | **+37.4%** |
| 销售净利率 | 12.04% | 15.74% | +3.7pp |

**三点硬结论**:
1. **扣非(+37.4%) > 归母(+36.4%)** → 高增长剔除非经常损益后反而更高 = 与金山(投资收益 19.4 亿污染净利)机制**完全相反**, 这是真实经营成果而非纸面利润。
2. **剪刀差来源 = margin 正常化 + 运营杠杆**: 25Q1 净利率 12.04% 是过去 6 期的**年内最低点**(25Q1 12.04 → 25H1 15.02 → 25Q3 15.59 → 25FY 16.68 → 26Q1 15.74)。+12% 营收叠加费用率正常化, 把利润放大到 +36%。这是**可持续的经营杠杆**, 不是会计花招。
3. **盈利质量干净(TTM 口径)**: OCF-TTM ≈ 248.6 亿 / 归母-TTM ≈ 149.4 亿 = **现金含量 166%**(经营现金流反超净利)。对照金山 2.7% 纸面利润、立讯单季 -193% 季节性假象, 海康是三者中唯一 TTM 现金含量真实健康的。

⚠️ 诚实标注的二阶疑点: 25Q1 是年内净利率低点 → 26Q1 的 +36% 含一部分**低基数红利**。这是待中报坐实的核心(见 §5 证伪条件)。

## 3. 反向验证(待验证①)——结论: 有安全垫但不极端
reverse_dcf, r=10% / N=5y / EPS-TTM=1.628 / 现价 30.24:

| 退出倍数 | 隐含 5 年净利复合增速 g |
|---|---|
| 16x | **13.3%** |
| 20x | 8.4% |
| 24x | 4.5% |

**公允价测算(若实际 g=15%)**: 退出 16x → 32.5(现价低估 7%) / 20x → 40.7(低估 26%) / 24x → 48.8(低估 38%)。

**硬结论**:
- 现价保守退出 16x 仅 price-in **13.3% 复合增速**, 而海康历史增速中枢 15-25%、当前单季净利 +36%。这是**正的安全垫**, 但不如宁德(现价隐含 15% vs 实际 +48.5%, 垫更厚)极端。
- **13.3% 是盈亏平衡隐含门槛**: 只要未来 5 年净利复合 ≥ 13%, 现价在保守倍数下就不贵。考虑安防主业增速放缓的现实, 13% 是个**够得着但不轻松**的门槛 → 海康是"合理偏低估"而非"白菜价"。
- 对照组: 工业富联现价隐含 25-30% 贴着实际无垫(高分位+负α); 海康隐含 13% 留垫(便宜区+利润加速)。**信号质量海康 > 工业富联**, 与上轮资本配置错配 note 的方向一致。

## 4. 地缘敞口量化(待验证③)——核心风险, 不可回避
主营构成(25FY, akshare 一手 zygc_em 按地区分类):
- **境内 58.22 亿元收入 / 占 62.9% / 毛利率 45.5%**
- **境外 34.29 亿元收入 / 占 37.1% / 毛利率 46.5%**(海外毛利率略高于境内)

**关键风险**: 海康自 2019 年起在**美国实体清单**, 海外 37.1% 收入(且毛利率更高)直接暴露于出口管制/关税升级。这正是 PE 被压到史 6% 分位的**结构性原因**——市场不是看不懂海康便宜, 而是给地缘尾部打了折。

**Polymarket 实查(地缘坐标)**:
- "US x China tariff agreement by June 30?" → **Yes 仅 4%**(成交 $28.8K)= 关税近期几乎必然延续
- "US x China tariff agreement by December 31?" → **Yes 75%**(成交 $16.8K)= 年底大概率缓和
- 读数: 近月地缘逆风持续(压制海外 37% leg), 年底有缓和期权。海康海外敞口比工业富联(AI server 出口)更直接绑制裁名单, 但**境内 62.9% 基本盘对关税近免疫**, 提供缓冲。

## 5. 可证伪论点 + 证伪条件(绑中报 8/31)
**论点**: 海康 18.6x 是市场对①安防/G端需求增速放缓 ②海外实体清单/关税敞口 两重担忧压低的倍数, 但利润加速(+36%扣非)+ 自身便宜区(史6%) + 干净现金流(TTM 166%)三者罕见共存 = 合理偏低估的优质成长, 非价值陷阱。

**证伪条件(任一触发则论点削弱)**:
1. 中报归母净利 YoY 跌破 +15%(隐含门槛 13% 之上无垫)→ 说明 26Q1 +36% 主要是 25Q1 低基数红利, 增速二阶导回落。
2. 中报销售净利率回落到 13% 以下 → margin 正常化是假象, 运营杠杆不可持续。
3. 海外收入 YoY 转负或占比骤降 → 实体清单/关税敞口实质恶化, 37% 高毛利 leg 受损。

**反方论据(抗确认偏误)**:
- 25Q1 净利率 12.04% 是年内低点 = +36% 含低基数成分, 单季一个数据点统计意义弱。
- 安防行业 G 端(政府/平安城市)需求与地方财政强相关, 财政紧约束下增长天花板可能比历史低。
- 海外实体清单是**结构性、不可逆**的折价, 不会因为一次中报 beat 就消失 → 18.6x 可能是"该有的折价"而非错杀。
