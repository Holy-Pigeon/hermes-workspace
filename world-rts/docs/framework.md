## 三、每格怎么填

每个 `(阵营 × 维度)` 单元格 = **① 硬指标原值 + 来源 ② 归一化到 0–100 的自给/强度分 ③ 趋势箭头（↑↓→）**。
归一化是命门：只有压到同一把尺子上，能源/粮食/半导体才能横向比，"最低分"才有意义。

## 四、瓶颈公式（正交四因子 · 含用户补丁：安全放大器）

> **瓶颈强度 = 物理缺口 × 战略相关性 × 填补难度 × 安全放大器**
>
> 读法：**缺多少 × 在乎不在乎补 × 能不能靠市场填 × 会不会不计成本砸钱**。四个因子彼此正交，各测一件事。

> ⚠️ **为什么不是"野心缺口"**：早期版本用 `(100−自给分) × 不可替代性 × 野心缺口`，把野心缺口定义为"目标水位 − 当前水位"。但"当前水位"在第一项已用过，野心缺口里又含一次——**同一个"缺"被乘两遍，缺口大的维度双重惩罚，排序失真**。故把它重定义为纯粹的"战略相关性权重"（剥掉当前水位），四因子正交。

### 项1 · 物理缺口 = (100 − 自给分) / 100　范围 0–1　【唯一纯硬数据项】
唯一完全由数据决定、不掺判断的一项。自给分归一化协议：
- **A 类（天然百分比：自给率/进口依赖度）** → 直接用。谷物自给率 85% = 85 分；进口依赖度 70% 取反 = 30 分。
- **B 类（非百分比连续量：水 m³/人、抚养比、军费/GDP）** → **锚点线性映射**（不用 min-max，否则"大家都缺"时失真）。每维定 `lo`(0分线)/`hi`(100分线) 两个物理锚点，`score = clip((x−lo)/(hi−lo)×100, 0, 100)`，方向不对取反。锚点取国际公认警戒线/标杆国（如水以 1700 m³/人国际缺水警戒线为低锚）。每维 lo/hi 灌数据时逐个定死写入文档，可复查可调。

### 项2 · 战略相关性权重　范围 0–1　【软·硬锚点】
测：这个维度是不是该阵营**核心野心的关键路径**？（芯片对中国=关键路径，对海湾=无关）
- 依据：官方战略文件（五年规划 / 国家安全战略 / 产业政策白皮书）是否列为优先项。
- 离散档：`0.2 无关 / 0.5 相关 / 0.8 重要 / 1.0 核心关键路径`
- **标注"基于政策文件判断"，非硬数据**（但依据是可查一手文件）。

### 项3 · 填补难度 = 不可替代性　范围 1–3　【半客观】
测：被卡断后能否靠**进口 / 替代品 / 库存**填上？能填 = 不算真瓶颈。
- 三个可查锚点：① 有几个替代供应来源？② 有无技术替代品？③ 库存/缓冲能顶多久？
- 离散档：`1.0 多来源易替代(普通大宗) / 1.5 有限来源 / 2.0 寡头或技术壁垒 / 3.0 单点卡死无替代(EUV/先进制程/淡水)`
- 供应来源数量可查（硬），综合成系数是判断——半客观。

### 项4 · 安全放大器（★ 用户补丁）　范围 1–3　【软·硬锚点】
测：该维度是否被该阵营**安全化**，决策函数是否已切到"不计成本"？
- 核心逻辑：**同样 60 分自给率，一旦被划入"国家安全/生存自主"，决策函数从"利润最大化"切到"生存/自主最大化"，无视投入产出比砸钱。** 不乘此因子会系统性低估被安全化领域的资金烈度——而那恰是政策资金最烈、最不敏感于估值、最肥的利润池。
- 依据：官方战略文件是否明确列为"国家安全"事项 + 是否配套**非市场化投入**（专项基金/举国体制/进口禁令）。
- 离散档：`1.0 纯商业 / 1.5 政策关注 / 2.0 战略安全 / 3.0 生存级(写入国安战略且配非市场化投入)`
- 按 `(阵营, 维度)` 二维赋值（非全局常数）。各阵营安全敏感清单（初判，灌数据时按一手文件核）：
  - 中国：半导体、粮食、能源、种子、关键矿产精炼 → 高
  - 美国：军事投射、先进制造回流、AI/算力 → 高
  - 欧洲：能源安全、再军事化 → 高
  - 印度：能源进口、粮食 → 高
  - 海湾：水、粮食、石油美元转科技树 → 高
  - 俄罗斯：军事、能源出口通道 → 高
- 标注"基于政策文件判断"，非硬数据。

### 输出与可靠性分级（纪律）
- **总分范围** = `[0,1] × [0,1] × [1,3] × [1,3]` = **0 到 9**，展示时线性拉到 0–100 高亮排序。
- **透明度铁律**：4 项里**只有项1 是纯硬数据**，项2/3/4 是"硬锚点 + 人工赋档"的半主观量。每格**分开存硬数据分和赋档分**，报告显式区分，绝不把主观档位伪装成硬数据。任何瓶颈结论都能回溯到"哪部分是数据、哪部分是判断"。

### 示意算例（占位数字，**非查证**，仅演示机制）
"中国·半导体"（下列全为占位假设，真数据灌库时查 SIA/海关）：

| 因子 | 取值(占位) | 依据 |
|---|---|---|
| 物理缺口 | (100−25)/100 = **0.75** | 假设自给分 25 |
| 战略相关性 | **1.0** | 十四五列核心攻关 |
| 填补难度 | **3.0** | EUV/先进制程单点卡死 |
| 安全放大器 | **3.0** | 举国体制+大基金，生存级 |
| **瓶颈强度** | 0.75×1.0×3.0×3.0 = **6.75 / 9** | 极高瓶颈 |

对照"美国·半导体"：战略相关性同高，但填补难度低（有台积电/三星盟友供给）、安全放大器中等 → 分数明显低于中国。这就是公式该有的区分力：不让两个都"重要"的格子糊成一片，而把"谁被卡到会不计成本砸钱"顶出来。

每个阵营自动吐出 1–2 个"绑定约束"，全程无人工指认——这就是"清晰全局视野"的成品形态。

## 五、这个模型会在哪骗你（主动对抗叙事谬误）

1. **RTS 是零和的，真实世界不是。** 现实里贸易可以正和；别把所有关系都误判成你死我活的战争框架。
2. **RTS 里资源→兵力是确定公式，现实里全是滞后+泄漏。** 政策砸钱 ≠ 公司赚钱，中间隔着执行、腐败、产能过剩、技术路线赌错。看到瓶颈 ≠ 找到能赚钱的公司。
3. **战争迷雾是真的。** 我们看到的每个阵营状态都有信息滞后，框架给方向不给精确点位。
