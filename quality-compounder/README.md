# 质量复利发现器 (quality-compounder)

## 它解决什么元层问题
整个发现侧此前**只有一个镜头**：`stock-discovery/tech_screener.py` 的 ⭐候选硬门槛是
**「PE 自身历史分位 < 35%」**——必须先便宜才进尽调漏斗。这是纯深度价值/均值回归视角。

但 SOUL 第一条投资哲学是段永平**「好生意 > 好价格」**、Nick Sleep 规模经济复利、可选权价值。
系统在哲学与实现上自相矛盾：嘴上"好生意优先"，发现入口却"好价格(便宜)优先"。

**后果**：一只 ROE 35% / 毛利 60% / 跑道很长但 PE 处自身史 50 分位（不贵不便宜）的伟大复利
机器，会被 tech_screener 判为「·中性」永不进漏斗。这是**锚定效应（锚在历史估值分位）在系统层
固化**，也解释了为什么研究库 18 篇 note、预测台账 13 条预测几乎全是"低分位陷阱 vs 错杀"同质范式。
巴菲特后期范式（以合理价买伟大公司，而非在历史最低估值抄底）这个系统**根本看不见**。

## 它是什么
tech_screener 的**正交镜头**：把锚从"历史估值低分位"换成"企业复利质量本身"。
- **价值漏斗**(tech_screener) 答：够不够便宜？
- **质量漏斗**(本器) 答：够不够好？
- 两漏斗殊途同归，都把候选喂给 `moat_scorecard`(护城河尽调) + `reverse_dcf`(反算隐含增速)。

## 判定逻辑（锚=质量，非估值）
- 💎 **高质量复利候选** = 复利硬门槛全过 且 PE 分位 < 90%(泡沫护栏)
  - ROE 年度 ≥ 18%
  - ROE 持久性(可得年报里 ≥15% 占比) ≥ 70%
  - 毛利率 ≥ 35%（定价权）
  - 净利率 ≥ 12%（排除杠杆/周转伪高 ROE）
  - 营收成长广度(最近5段同比为正占比) ≥ 60%（再投资跑道）
  - TTM 现金含量(OCF/归母) ≥ 60%（真钱非纸面）
- 🌱 **准复利** = 仅差 1 项（列出差哪项，供人工跟）
- 💰 **优质但已贵** = 门槛全过但 PE 分位 ≥ 90%（好公司但当前价不合理，入观察不入尽调）
- ·  不符合

## 与 tech_screener 协同
同一标的若 tech_screener 判 ⭐(便宜) + 本器判 💎(优质) = **双镜头共振 = 最高优先尽调**。
本器 💎 但 tech_screener ·中性 = "好公司但不便宜"——正是本器补的盲区。

> **正交性诚实声明（2026-06-20 经 signal-orthogonality 审计）**：⭐+💎"双镜头共振"
> **不是两条完全独立的印证**。signal_orthogonality 实测 tech_screener × quality_compounder
> 共享根输入 `{price, income_statement}`（Jaccard=0.50）。本器相对价值镜头**真正独立的增量**
> 是 `cash_flow`(TTM现金含量) + `balance_sheet`(净利率排杠杆) 两维，**而非"又一票独立看好"**。
> 引用共振时应按【独立根输入数】计权，避免把 0.5 重叠当 2 重印证制造虚假信心。
> 本器已登记 signal_registry.json，纳入元层正交审计。

## 用法
```
/opt/homebrew/bin/python3 quality_screener.py            # 全白名单扫描
/opt/homebrew/bin/python3 quality_screener.py --quiet    # 只在有💎候选时输出(可挂cron)
/opt/homebrew/bin/python3 quality_screener.py --symbol 600519
```

## 数据诚实
- 全部一手 akshare（`stock_financial_abstract` + `stock_value_em`），复用 tech_screener 同款
  ROE年度(最近1231)/TTM现金含量/净利率口径，端口失败明确跳过绝不编造。
- 复用 `stock-discovery/watchlist.json`（同一能力圈，不另维护避免漂移）。
- ROE 持久性仅基于可得年报年份；成长广度受基数/季节性影响；非买卖指令，💎仅代表值得深研。

## 首跑实测(2026-06-19, 10 只白名单)
💎 茅台(双镜头共振) | 🌱 宁德(差毛利率)/迈瑞(差成长广度)/海康(差ROE 17.3%) | 其余不符合。
正交性验证：迈瑞 PE 1%分位(价值镜头最爱)却被质量镜头拦在🌱(再投资跑道存疑)，证明两镜头不冗余。
